SSH Terminal Workspace

Welcome to your command center. Connect to multiple servers simultaneously and monitor live resource usage (CPU, RAM, Disk) directly in the title bar. Use the Upload (📤) and Download (📥) buttons for quick SFTP file transfers. You can reorder terminals using the arrows, or use Horizontal Scroll to infinitely add full-sized terminals side-by-side without them ever getting squished. You can also compare specific tabs using Split View, and save your entire layout to Session History to instantly restore it later.
